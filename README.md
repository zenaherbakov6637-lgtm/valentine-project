📖 Описание проекта
FLOWER_HACK — это романтическая интерактивная веб-страница, стилизованная под интерфейс «взлома». Проект создан для оригинального признания в любви через необычный цифровой формат.
При открытии пользователь видит:
🔓 Анимацию прогресс-бара «взлома»
🎮 Интерактивные кнопки управления
💖 Финальное сообщение «Я ТЕБЯ ЛЮБЛЮ»
